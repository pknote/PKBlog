Github 上有大量开源资源供我们学习，但国内访问和下载困难的情况一直存在。很多人也没有稳定的科学上网手段，那么以下这些方法可以帮助你顺利访问 Github，高频使用的同学还是建议采用 **[友情链接](https://nav.pknote.top/)** 推荐的科学上网工具。

## 加速镜像

1. **[KGitHub](https://help.kgithub.com/questions/)** ：查看代码仅需在 `github.com` 前加上 `kk` 即可。
2. 将原 Github 链接的 `github.com` 部分修改为 `githubfast.com` ，可以加速访问。
3. **github.hscsec.cn** 是一个国内加速项目，只需要把 `github.com` 替换为 `github.hscsec.cn` 就可以解决访问问题。


## 工具插件

1. [**Watt Toolkit**](https://steampp.net/) ，中文叫瓦特工具箱，是一个开源跨平台的多功能 Steam 工具箱，具备优化 Github 访问功能。
2. **[SwitchHosts](https://github.com/oldj/SwitchHosts)** ，是一个管理 hosts 文件的应用，基于 Electron 、React、Jotai 、Chakra UI、CodeMirror 等技术开发。

## 其它汇总

**[Github Hosts](https://hosts.gitcdn.top)** 每日自动更新。

访问 https://hosts.gitcdn.top/hosts.txt ，将其全部内容粘贴到你的hosts文件中，即可。

**添加hosts**

- Linux / MacOS hosts路径：`/etc/hosts`
- Windows hosts路径：`C:\Windows\System32\drivers\etc\hosts`

**刷新生效**

- Linux: /etc/init.d/network restart
- Windows: ipconfig /flushdns
- Macos: sudo killall -HUP mDNSResponder

可以手动修改hosts，或用软件 [**SwitchHosts**](https://github.com/oldj/SwitchHosts) 修改，也可使用该服务官方提供的 **[客户端](https://kkgithub.com/Licoy/fetch-github-hosts/releases)** 。
