很多同学用 Google Chrome 浏览器在网上看到很多有用的小知识或者精美的落地页，页面上并没有明显的下载按钮，又没有像其他浏览器自带整页截图插件的时候，就很苦恼， Google Chrome 浏览器怎么截取网页长图呢？

![](/images/202404011002096.webp)

今天胖氪就来带大家不用插件，几个快捷键就可以实现网页长截图。

1. **打开 Google Chrome 浏览器，键盘按下快捷键；**

```
Windows ：Ctrl + Shift + I 
MacOS ：Command + Shift + C
```

即可打开 **开发者工具** 窗口。

![](/images/202404011002163.webp)

2. **接下来再按下快捷键；**

```
Windows ：Ctrl + Shift + P
MacOS ： Command + Shift + P
```

3. **出现 Google Chrome 浏览器指令选单窗口，输入指令：`Capture full size`**
   **`screenshot`，回车即可自动保存长截图到浏览器指定的下载目录。**

![](/images/202404011003048.webp)

4. **其他指令还有：**

```
Capture full size screenshot ：长截图

Capture screenshot ：一般截图

Capture node screenshot ：预览截图

Capture area screenshot ：自选区域截图
```

大家可以按需使用，我的习惯是只记住长截图指令就可以了，所有网页都截长图，后面有需要局部的可以在长图上面裁剪就可以了。
