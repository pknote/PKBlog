作为一名天天和命令行打交道的人，我太懂那种痛了——

每次 `ls` 出来的黑白文字看半天找不到文件，`cat` 一个代码文件乱成一团，`git log` 更是让人眼花缭乱……

说实话，命令行本来可以很美的！

今天胖氪就给大家推荐 **5款开源免费的命令行效率神器**，让你的终端体验直接上一个档次！


## 1️⃣ eza — 升级版 ls

![eza — 升级版 ls](/images/16801.webp)

`ls` 是你每天必用的命令吧？但默认的输出实在太朴素了。

**eza** 是一个 modern 版 `ls`，支持：

- 🦋 彩色输出，文件类型一目了然
- 📁 目录树形显示
- 🏷️ Git 状态集成（哪个文件被修改了清晰可见）
- 📊 文件大小人类可读

**传送门**：https://github.com/eza-community/eza

```bash
# 一键安装（Linux）
sudo apt install eza

# 或者用 cargo
cargo install eza
```


## 2️⃣ bat — 升级版 cat

![bat — 升级版 cat](/images/16802.webp)

`cat` 虽好，但看代码的时候完全没有高亮，文件大了还直接刷屏。

**bat** 就是来解决这个问题的：
- 🎨 代码语法高亮
- 📜 自动分页，超大文件也不怕
- 🔍 行号显示
- 💾 还能当 `cat` 的完全替代品

写代码的时候再也不用"盲看"了，谁用谁香！

**传送门**：https://github.com/sharkdp/bat

```bash
# Linux
sudo apt install bat

# 或者
cargo install bat
```


## 3️⃣ fzf — 命令行模糊搜索神器

![fzf — 命令行模糊搜索神器](/images/16803.webp)

这绝对是今天最重磅的一款！

只要装上 fzf，你的终端就能实现：
- 🔍 **模糊搜索文件** — 输入几个字母就能找到匹配的文件
- 📝 **命令历史搜索** — 忘记上次跑的什么命令？一秒找到
- 🔄 **智能补全** — 各种命令的参数自动补全

配合 vim 或其他工具更是如虎添翼，用过的都说回不去了！

**传送门**：https://github.com/junegunn/fzf

```bash
# 安装
git clone --depth 1 https://github.com/junegunn/fzf.git ~/.fzf
~/.fzf/install
```


## 4️⃣ delta — 让 git log 好看10倍

![delta — 让 git log 好看10倍](/images/16804.webp)

天天看 `git log` 是不是头疼？密密麻麻的文字根本找不到重点。

**delta** 是一个 git pager，能让你：
- 🌈 语法高亮 git diff/output
- 📊 清晰展示文件变更
- 📝 side-by-side 对比模式
- 🎯 单词级别差异高亮

装了 delta 之后，看 git 输出变成了一种享受（真的）！

**传送门**：https://github.com/dandavison/delta

```bash
# 下载二进制
curl -LSs https://github.com/dandavison/delta/releases/latest/download/delta-musl-static-x86_64.tar.gz | tar xz
sudo mv delta-musl-static-x86_64.tar.gz /usr/local/bin/delta
```


## 5️⃣ tldr — 简化版 man 手册

![tldr — 简化版 man 手册](/images/16805.webp)

`man` 手册虽全，但太啰嗦了有没有！看半天找不到重点。

**tldr**（Too Long; Didn't Read）用一句话解释每个命令，还带实用示例：

```
$ tldr tar

# tar
Archiving utility.
- Create an archive from files:
  tar -cf archive.tar file1 file2

- Extract archive:
  tar -xf archive.tar
```

妈妈再也不用担心我记不住命令了！

**传送门**：https://github.com/tldr-pages/tldr

```bash
# Node.js 用户
npm install -g tldr

# 或者用 pip
pip install tldr
```
