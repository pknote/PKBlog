一款开源免费并且可以自托管的轻量笔记服务，让你可以随时随地记录和分享你的想法。

**项目特点：**

  1. 开源并且永久免费；
  2. 支持几秒钟实现Docker自托管部署；
  3. 笔记支持Markdown，让你更专注文字内容的构思；
  4. 支持自定义并且能简单轻松的分享你的笔记；
  5. 拥有第三方的RESTful API服务支持。

**Docker部署：**

```dockerfile
docker run -d --name memos -p 5230:5230 -v ~/.memos/:/var/opt/memos neosmemo/memos:stable
```
