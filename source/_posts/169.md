作为一个天天要和英文文档打交道的人，我太懂那种痛了——

装了 N 个翻译软件来回切换，Google 翻译不准、DeepL 收费、ChatGPT 还要翻墙……

直到我在 GitHub 上发现了这款开源工具，直接解决了所有痛点！

今天胖氪要推荐的就是 **STranslate**，一款 WPF 开发的高颜值翻译+OCR 工具，用过一次就离不开！

![STranslate](/images/16901.webp)


## 🔥 核心功能

### 1️⃣ 多源翻译 — 一次查询，10家结果对比

这绝对是 STranslate 最核心的功能！

集成翻译源包括：
- OpenAI (GPT 系列)
- Google Translate
- DeepL
- Gemini
- 百度翻译
- 阿里翻译
- ……还在持续增加

也就是说，你输入一段文字，**一次性返回多家翻译结果**，哪个准确一眼看出！

再也不用来回切换 App 比对结果了，一个窗口全搞定！


### 2️⃣ 划词翻译 + 快捷键 — 选中即翻，不用切换窗口

以前翻译一段文字，要：
1. 复制文本
2. 切换到翻译软件
3. 粘贴
4. 复制结果
5. 粘贴回去

STranslate 直接**划词就翻**！选中文字，按个快捷键，弹窗直接出结果，整个过程不到 1 秒！

而且还支持**静默翻译模式**，弹窗自动消失，不打断你的工作节奏，简直是效率杀手！


### 3️⃣ 智能 OCR — 截图翻译，看图识字

这个功能简直是留学生的救命稻草！

内置**微信 OCR 引擎**，不用联网就能精准识别图片文字。

最绝的是：**截图 → 识别 → 翻译 → 原文位置直接呈现译文**，所见即所得！

看英文文档、PDF、海报，再也不用一个个字敲了，直接拍照翻译！


### 4️⃣ 插件系统 — 功能无限扩展

STranslate 基于 .NET 架构，支持插件扩展！

官方和社区已经提供了不少插件：
- 🔊 TTS 语音朗读（支持多国语言，语速可调）
- 📝 AI 润色/总结（接入大模型）
- 🌐 自定义翻译源

后期想加什么功能，插件市场逛一逛，说不定就有人帮你写好了！


### 5️⃣ 生词本 — 背单词神器 🧠

这个功能我没想到会被狠狠种草！

阅读时遇到的生词，一键收藏到生词本，**支持本地备份和复习回顾**。

学英语最难的就是坚持，这个功能让翻译和背词形成闭环，实用到离谱！


### 6️⃣ 其他细节 — 全是加分项

- 🎨 明暗主题切换，颜值在线
- 🔢 二维码识别
- 📜 历史记录查看
- ⚙️ 全局热键自定义
- 📡 外部 API 调用

## 总结一下

说实话，这类翻译工具我之前也试过不少，但要么收费、要么有广告、要么功能单一。

STranslate 真正吸引我的是两点：
1. **开源免费**，没有乱七八糟的弹窗
2. **多源对比**，一个窗口解决选择困难症

如果你也在找一款好用的翻译工具，建议试试这款！
